mnml is one of three small Rust tools by Chris McLennan. They each
solve a different problem — an IDE, a terminal, a DJ rig — but they
ship from the same workshop, share the same design language (the
charcoal rounded-square icon, the `>` shell-prompt wordmark, the
monospace identity), and they talk to each other via the
`tmnl-protocol` cell-streaming layer. Run them standalone in any
terminal, or run them together: tmnl hosts mnml as a native tab, and
mnml hosts mixr in a side panel.

## How they fit together

They share more than a coat of paint:

- **Shared design language.** Charcoal rounded-square icons, a
  `>` prompt accent in each app's color (cool blue for mnml, magenta
  for tmnl, gold for mixr), monospace wordmarks. Same family settings
  UI idiom — a sectioned scrollable list with `▸ <label>: [active] /
  other  *` rows, the same keybindings (`←→` adjust, `↑↓` move, `r`
  reset row, `R` reset all, `Enter` save, `Esc` cancel).
- **Shared cell-streaming protocol.** [`tmnl-protocol`](https://github.com/chris-mclennan/tmnl-protocol)
  is a small wire format for "send me a grid of styled cells." tmnl
  hosts mnml as a native tab using it; mnml hosts mixr in a side
  panel using it; mnml's `:host.launch <binary>` opens any
  `tmnl-protocol`-speaking binary as a `Pane::BlitHost`. One protocol,
  three integration shapes.
- **Pty-fd handoff.** Inside tmnl, `:tmnl.pop-pty` in mnml transfers
  the focused terminal pane's pty file descriptor to tmnl via
  `SCM_RIGHTS`. The shell keeps running; it just becomes a sibling
  native tab instead of a nested pane. (Unix only.)
- **Shared `fim-engine`.** The same in-process FIM (fill-in-the-middle)
  Rust crate that powers tmnl's `⌘I` completion also powers mnml's
  optional offline inline suggestions. One quantized model download
  serves both apps.

You don't need all three. mnml is happy in iTerm2, Alacritty, kitty,
or Windows Terminal. mixr runs standalone too. tmnl is just `$SHELL`
in a window if you never open the native tabs. But run them together
and the seams disappear.
