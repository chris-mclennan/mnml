Or paste the shell installer for your platform:

```sh
# macOS / Linux
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/chris-mclennan/mnml/releases/latest/download/mnml-rs-installer.sh | sh
```

```powershell
# Windows
irm https://github.com/chris-mclennan/mnml/releases/latest/download/mnml-rs-installer.ps1 | iex
```

## Platform details

<CardGrid>
  <Card title="macOS" icon="apple">
    `.pkg` installer drops `mnml` on `PATH`. Per-arch — Apple Silicon or Intel.

    Or grab the `.tar.xz` from the [latest release](https://github.com/chris-mclennan/mnml/releases/latest) for manual install.

    Requires macOS 10.14 (Mojave) or newer. A `.dmg` with a dock-icon install is in the queue.
  </Card>
  <Card title="Linux" icon="linux">
    Shell installer drops `mnml` on `PATH`:

    ```sh
    curl -LsSf https://github.com/chris-mclennan/mnml/releases/latest/download/mnml-rs-installer.sh | sh
    ```

    Or grab the `.tar.xz` for your arch from the [latest release](https://github.com/chris-mclennan/mnml/releases/latest).
  </Card>
  <Card title="Windows" icon="seti:windows">
    Double-click the `.msi` from the [latest release](https://github.com/chris-mclennan/mnml/releases/latest), or PowerShell:

    ```powershell
    irm https://github.com/chris-mclennan/mnml/releases/latest/download/mnml-rs-installer.ps1 | iex
    ```
  </Card>
  <Card title="Cargo" icon="rocket">
    Build from crates.io (slow — requires the Rust toolchain):

    ```sh
    cargo install mnml-rs
    ```

    The crate is `mnml-rs`; the binary it installs is `mnml`.
  </Card>
</CardGrid>

## Build from source

```sh
git clone https://github.com/chris-mclennan/mnml
cd mnml
cargo build --release
./target/release/mnml
```

## Nerd Font

A [Nerd Font](https://www.nerdfonts.com/) is recommended for devicons and powerline glyphs. JetBrainsMono Nerd Font is a good default. Without a Nerd Font, run `mnml --ascii` (or set `[ui] ascii_icons = true` in your config) for a plain-text fallback.

## Verify

```sh
mnml --version
```
