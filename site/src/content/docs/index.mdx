## What is mnml

mnml is a terminal IDE in the NvChad aesthetic ŌĆö file-tree rail, tabline, powerline statusline, Nerd-Font devicons, tree-sitter highlighting ŌĆö built on one design decision that runs through everything: **a pluggable input layer.** Vim modal editing and a modeless VS Code-style keymap are both first-class, both fully remappable, and swappable at runtime. The editor, buffer, and render layers never branch on which one is active.

Install it as a proper macOS `.app` (DMG, drag to Applications, click the dock icon and mnml opens in Terminal.app) or as a CLI on any platform.
