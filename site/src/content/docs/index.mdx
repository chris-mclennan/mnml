## What is mnml

mnml is a terminal IDE in the NvChad aesthetic — file-tree rail, tabline, powerline statusline, Nerd-Font devicons, tree-sitter highlighting — built on one design decision that runs through everything: **a pluggable input layer.** Vim modal editing and a modeless VS Code-style keymap are both first-class, both fully remappable, and swappable at runtime. The editor, buffer, and render layers never branch on which one is active.

## The family

mnml is one of three small Rust tools by Chris McLennan:

- **mnml** — this IDE
- **[tmnl](https://tmnl.sh)** — a GPU-rendered terminal that lets mnml run as a native tab instead of nested in some other terminal emulator
- **[mixr](https://mixr.sh)** — a terminal DJ app

You can run mnml inside any terminal that supports modern escape codes. Inside tmnl specifically, mnml gets native-tab integration via the `tmnl-protocol` cell-streaming layer — plus a pty-fd handoff (`:tmnl.pop-pty`) that lets you pop a running shell pane out into a sibling native tab without killing the child process.
