## What is mnml

mnml is a terminal IDE in the NvChad aesthetic — file-tree rail, tabline, powerline statusline, Nerd-Font devicons, tree-sitter highlighting — built on one design decision that runs through everything: **a pluggable input layer.** Vim modal editing and a modeless VS Code-style keymap are both first-class, both fully remappable, and swappable at runtime. The editor, buffer, and render layers never branch on which one is active.

## Headless + scriptable

`mnml --headless` renders to a virtual screen driven over a file-IPC channel; a line-based `.test` format runs end-to-end UI tests against the same `App` the terminal uses. Same architecture powers both — no separate test harness, no mode drift.

## The family

mnml is one of three small Rust tools that work together:

- **mnml** — this IDE
- **[tmnl](https://tmnl.sh)** — a GPU-rendered terminal that lets mnml run as a native tab instead of nested in some other terminal emulator
- **[mixr](https://mixr.sh)** — a music app, also tmnl-native

You can run mnml inside any terminal that supports modern escape codes. Inside tmnl specifically, mnml gets native-tab integration via the `tmnl-protocol` cell-streaming layer.
